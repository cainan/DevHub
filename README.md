# DevHub
Project based on Alura-7DaysOfCode

This Android project can fetch information about a GitHub user via API

## Technologies Used

- Kotlin
- Jetpack Compose
- Retrofit
- Moshi

## How to Use

Clone the project, open it in Android Studio, and run it on an Android device or emulator.
